# Changelog

All notable changes to this project will be documented in this file.

This project follows a simple semantic versioning strategy and tracks the evolution from the original JavaScript implementation to the modern TypeScript version.

---

## [2.0.0] — TypeScript Migration (Current Version)

### Added
- Full migration from JavaScript to TypeScript
- Strong typing for task entities
- Improved project structure
- TypeScript configuration (tsconfig.json)
- Build process for compiling TypeScript to JavaScript
- Production-ready entry point

### Changed
- Refactored application architecture
- Improved variable naming and code readability
- Updated scripts for TypeScript execution
- Main branch switched to TypeScript implementation

### Removed
- Legacy JavaScript-only patterns
- Implicit `any` usage
- Unsafe dynamic typing logic

---

## [1.0.0] — Initial JavaScript Release (Legacy Version)

### Added
- Console-based Task Manager
- Basic CRUD operations for tasks
- Simple CLI interaction
- Initial project structure
- Node.js runtime support

### Notes
- This version is preserved in the `legacy-js` branch
- Kept for historical reference and learning purposes

---

## Versioning Strategy

- `1.x.x` → JavaScript legacy implementation
- `2.x.x` → TypeScript implementation
- Major versions indicate architecture or language changes
- Minor versions indicate feature additions
- Patch versions indicate bug fixes
